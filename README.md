# Otternel

Successor of [ServerSentinel](https://github.com/Corentin-cott/ServerSentinel). Otternel is a Rust service that stores player game data and monitors server logs. 

# Functionalities

- Interact with server logs based on triggers
- Get Minecraft & Palworld players statistics *(WIP)*
- Periodically regulate Periodically on launched servers *(WIP)*
